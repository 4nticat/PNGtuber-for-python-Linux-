PNGtuber for Python (Linux)

This is a lightweight Python-based program designed for Linux users who want an efficient PNGtuber avatar for streaming or recording. This tool is built to be minimalist and highly customizable for technical environments.
📁 Project Structure

    pngtuber.py: The main script that handles the logic and image switching.

    abierta.png: Avatar image with the mouth open.

    abierta_p.png: Avatar image with the mouth open and eyes closed (blinking).

    cerrada.png: Avatar image with the mouth closed.

    cerrada_p.png: Avatar image with the mouth closed and eyes closed.

🚀 Key Features

    Linux Optimized: Developed specifically to run smoothly on Linux distributions.

    Resource Efficient: Low overhead, making it ideal for setups where performance is a priority.

    Easy Customization: Swap the .png files or tweak the Python code to fit your specific needs
